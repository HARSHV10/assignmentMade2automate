# Made2Automate Display Project

## Description
This project is developed for Made2Automate, a company that specializes in automation solutions. The project consists of two main folders: `backend` and `frontend`.

### Backend
The `backend` folder contains the server and REST API responsible for managing the company's details and handling the upload of electronic products.

#### Files and Folders
- **server.js**: Backend server configuration file.
- **request.rest**: Use this file with tools like Visual Studio Code's REST Client extension to test REST API endpoints.
- *Other backend-related files*

### Frontend
The `frontend` folder is structured with a `components` subfolder, which includes components for the user interface, such as a navigation bar.

#### Files and Folders
- **components**: Folder containing various frontend components, including the navbar and others.

## Getting Started

### Prerequisites
Ensure the following are installed:
- Node.js
- npm (Node Package Manager)

### Installation

#### Backend
1. Navigate to the `backend` folder in the terminal.
2. Run `npm install` to install the required dependencies.
3. Start the backend server by running `node server.js`.

#### Frontend
1. Navigate to the `frontend` folder in the terminal.
2. Run `npm install` to install the required dependencies.
3. Start the frontend development server by running `npm start`.

## Usage

### Backend
- Modify the `server.js` file for any additional server configurations.
- Helper for all helper functions in the exrpess route
- databse folder to store collections and initialize database 
- Utilize the `request.rest` file to test the REST API endpoints.

### Frontend
- Customize components in the `frontend/components` folder to suit the UI requirements.
- Enhance or modify components to meet specific needs.
